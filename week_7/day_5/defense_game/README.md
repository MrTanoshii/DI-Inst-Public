# Defense Game

Shoot down the incoming asteroids to save your planet!

## Controls

|   Hotkey    | Description |
| :---------: | ----------- |
| ARROW_LEFT  | Move left   |
| ARROW_RIGHT | Move right  |
|   ESCAPE    | Quit game   |
|  SPACEBAR   | Shoot       |
